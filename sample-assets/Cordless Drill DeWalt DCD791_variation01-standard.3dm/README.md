## Purpose

This sample CAD asset is a good showcase for a simple parametric object with a few challenges:

1. Complex fillets and solid parts which can be challenging for tessellation process.  

2. Interior Parts (battery charger etc.).  

3. Extruded text as part of the CAD surfaces. This can be very challenging for 3D processing and optimization after initial tessellation.
