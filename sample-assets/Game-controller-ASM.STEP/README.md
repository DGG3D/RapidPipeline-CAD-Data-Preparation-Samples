## Description

This was designed by Mark Biasotti at SolidWorks, and he has made the file available for download. There is no written tutorial to go with it, but just reviewing the files and how they were created should be beneficial. The model shows both master modeling techniques, and was created with surface modeling tools.

## Purpose

This simple example CAD asset presents a good basis for demonstrating certain 3D processing and optimization features (e.g. remeshing). The overall shapes are very organic and should therefore behave quite differently compared to other more mechanic and hard-surface style data.
